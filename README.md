# Машинное обучение и анализ данных

Репозиторий с учебными задачами по машинному обучению (ВМК МГУ, 5-6 семестры)

В ходе курса решались задачи, связанные с анализом данных, обучением моделей и оценкой их качества. Основное внимание уделялось не только применению алгоритмов, но и пониманию данных и интерпретации результатов.

## Что делалось

- обработка и анализ данных (NumPy, Pandas)
- визуализация и исследование данных
- подготовка признаков и масштабирование
- обучение моделей
- подбор гиперпараметров
- оценка качества (MAE, accuracy и др.)
- сравнение моделей и анализ результатов

## Используемые методы

- kNN
- линейные модели
- SVM
- деревья решений
- ансамбли и градиентный бустинг

## Стек

Python, Jupyter Notebook, NumPy, Pandas, Matplotlib, scikit-learn
